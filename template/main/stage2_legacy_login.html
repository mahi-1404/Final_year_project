Stage 2: The Echo Chamber

"The legacy system reflects everything you say back to you. Perhaps it reflects too much. Can you make it speak in ways it shouldn't?"

Hint: XSS (Cross-Site Scripting) or SSTI (Server-Side Template Injection)

Recommended Fix — XSS / SSTI

Use strict output encoding on all user-supplied data before rendering it in HTML. Implement a Content Security Policy (CSP). Never evaluate template expressions built from user input — use sandboxed or logic-less templates and whitelist-based input validation.
