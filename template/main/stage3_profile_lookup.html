Stage 3: The XML Gateway

"The system accepts XML data for profile lookup. It trusts external entities... perhaps too much. Can you make it read things it shouldn't?"

Hint: XXE (XML External Entity Injection)

Recommended Fix — XXE (XML External Entity)

Disable external entity and DTD processing in your XML parser (e.g., set FEATURE_EXTERNAL_GENERAL_ENTITIES to false). Use a safe XML library or JSON instead of XML where possible. Never parse untrusted XML with default parser settings.
