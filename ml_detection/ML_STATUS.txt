## 💡 Quick Usage

### Basic Integration
```python
from ml_detection.flask_integration import detector_manager

result = detector_manager.check_input(user_input)
if result['is_malicious']:
    return f"Access Denied: {result['color']} {result['attack_type']}", 403
```

### Available Methods
- `check_input(input_text)` - Full detection result
- `is_malicious(input_text)` - Boolean check
- `get_color(input_text)` - Get 🔴 RED or 🟢 GREEN
- `get_status(input_text)` - Get "MALICIOUS" or "SAFE"
