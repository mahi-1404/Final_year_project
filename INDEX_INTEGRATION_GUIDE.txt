🚀 HOW IT WORKS

Step 1: User enters username/password in index.html
   ↓
Step 2: JavaScript detects blur event (user leaves field)
   ↓
Step 3: AJAX request sent to /ml_detect endpoint
   ↓
Step 4: Flask calls detector_manager.check_input()
   ↓
Step 5: ML model analyzes input (100% accuracy)
   ↓
Step 6: Response returned with:
   - color: 🟢 GREEN (safe) or 🔴 RED (malicious)
   - status: SAFE or MALICIOUS
   - attack_type: sql_injection, xss, or None
   - confidence: 87-100%
   ↓
Step 7: JavaScript updates input status indicator
   ↓
Step 8: On form submit:
   - If RED: Show sci-fi access denied screen
   - If GREEN: Proceed to /check endpoint

⚙️ CONFIGURATION

To modify detection behavior, edit in index.html:

1. Detection Trigger Point:
   • Currently: On blur (when user leaves field)
   • Can change to: On input, on change, etc.

2. Debounce Detection:
   • Add setTimeout to prevent excessive API calls
   • Recommended: 500-1000ms delay

3. Auto-submit Delay:
   • Currently: 1000ms after GREEN
   • Modify in setTimeout() for faster/slower response

4. Access Denied Duration:
   • Currently: 5000ms (5 seconds)
   • Modify: overlay.remove() timeout value
