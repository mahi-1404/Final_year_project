🎯 HOW THE INTEGRATION WORKS
═════════════════════════════════════════════════════════════════════════════

User enters username/password in login form
          ↓
    (blur event fired)
          ↓
JavaScript sends AJAX POST to /ml_detect
    {"input": "user_value"}
          ↓
Flask endpoint /ml_detect receives request
          ↓
detector_manager.check_input() analyzes input
          ↓
ML Model returns result:
{
    "color": "🟢 GREEN" or "🔴 RED",
    "status": "SAFE" or "MALICIOUS",
    "attack_type": "sql_injection", "xss", or null,
    "confidence": 0.87 - 1.00
}
          ↓
JavaScript updates UI with status indicator
          ↓
On form submit:
  • If 🟢 GREEN: Show "Input validated" → Proceed to /check
  • If 🔴 RED: Show "Access Denied" screen → Block login
